# ACK Advanced Bookmark Manager - 智能知识图谱书签管理器

ACK (Advanced Chrome Knowledge) 是一个下一代的 Chrome 书签管理扩展，它不仅仅是一个书签列表，更是一个通过 AI 驱动的个人知识图谱系统。

## 🌟 核心功能

### 🧠 AI 驱动的知识图谱
- **全量自动分析**：一键将成百上千个书签交由 AI 分析，提取主题分类并建立语义关联。
- **可视化宇宙**：基于 **D3.js** 开发的动态力导向图，让您的书签像繁星一样连接成片。
- **实时增量更新**：识别最新添加或修改的书签，秒级完成图谱补全，无需重新生成。
- **自动分类系统**：AI 自动为您的书签打标签（技术、资源、新闻、娱乐等），并分配精美的色彩方案。

### 🚀 工业级分析引擎
- **断点续传机制**：支持超大规模书签库（1000+）分批分析。即便中断、浏览器关闭或 API 报错，下次打开可完美恢复进度。
- **实时渲染技术**：分析过程中节点会动态“坠入”图谱，实现所见即所得的进度反馈。
- **多模型支持**：原生支持 **DeepSeek**、**硅基流动 (SiliconFlow)**、**OpenAI** 以及 **Anthropic** (支持 Claude 系列及 Extended Thinking 模型)。
- **极致稳定性**：采用前端直连 API 架构，绕过 Chrome Background 超时限制，确保长达数小时的分析任务不掉线。

### 📂 基础管理功能
- **树形视图**：清晰展示原生书签目录结构。
- **一键收藏**：快速抓取当前页面元数据。
- **智能搜索**：毫秒级的全文检索体验。
- **统计面板**：按年份、按目录统计书签增长趋势。

## 🛠️ 安装方法

1. 下载本项目代码。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的 **"开发者模式"**。
4. 点击 **"加载已解压的扩展程序"**，选择本项目文件夹。

## ⚙️ 快速配置

1. 点击扩展图标并进入 **"设置"**。
2. 在 **"AI 接口配置"** 中选择您的服务商（推荐 DeepSeek 或 硅基流动）。
3. 输入您的 **API Key**。
4. 点击 **"测试连接"**。
5. 前往 **"知识图谱"** 页面，点击 **"全量重新分析"**，开始构建您的个人知识宇宙。

```

│   └── settings.js       # 配置管理

├── graph.html            # 知识图谱展示大屏

└── README.md
```

## 🔒 权限与安全
- **unlimitedStorage**: 用于存储大规模知识图谱数据（Node/Edges）。
- **bookmarks**: 管理原生书签交互。
- **AI 隐私**: 我们仅发送书签的标题和域名给 AI 进行分类，不涉及敏感页面内容。API Key 存储在您的本地 `chrome.storage.sync` 中。
