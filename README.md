# BookmarkManager - 高级书签管理器

一个功能强大的Chrome浏览器扩展，用于增强和接管浏览器原生的书签管理功能。

## 🚀 功能特性

- **📚 书签展示**: 以树形结构清晰展示所有书签和文件夹
- **⚡ 一键收藏**: 快速收藏当前活动标签页
- **🔔 智能通知**: 书签操作完成后自动推送通知
- **🎨 现代设计**: 蓝色主题的现代化界面设计
- **📱 响应式布局**: 300px固定宽度，适配各种屏幕

## 🛠️ 安装方法

### 开发者模式安装

1. 打开Chrome浏览器，在地址栏输入 `chrome://extensions/` 并回车
2. 在页面右上角，打开 **"开发者模式"** 开关
3. 点击左上角的 **"加载已解压的扩展程序"** 按钮
4. 选择项目根目录（包含manifest.json的文件夹）
5. 点击"确定"完成安装

### 验证安装

安装成功后，您会看到：
- 浏览器工具栏出现扩展图标
- 收到欢迎通知消息
- 点击图标可打开书签管理界面

## 📖 使用指南

### 基本操作

1. **查看书签**: 点击工具栏图标打开弹出窗口
2. **浏览书签**: 在树形结构中浏览所有书签和文件夹
3. **打开书签**: 点击书签链接在新标签页中打开
4. **添加书签**: 点击"一键收藏当前页面"按钮

### 高级功能

- **自动通知**: 书签创建、删除、修改时自动推送通知
- **事件监听**: 后台服务持续监听书签变化
- **错误处理**: 智能处理各种异常情况

## 🔧 技术架构

### 核心技术

- **前端**: 原生HTML + CSS + JavaScript
- **扩展框架**: Chrome Extension Manifest V3
- **后台服务**: Service Worker

### 主要API

- `chrome.bookmarks`: 书签管理API
- `chrome.tabs`: 标签页管理API
- `chrome.notifications`: 通知推送API
- `chrome.runtime`: 运行时消息传递API

## 🔒 权限说明

扩展需要以下权限：

- **bookmarks**: 读取和管理用户书签
- **activeTab**: 获取当前活动标签页信息
- **notifications**: 发送桌面通知

## 🐛 故障排除

### 常见问题

1. **扩展无法加载**
   - 检查manifest.json语法是否正确
   - 确认所有文件路径存在

2. **书签无法显示**
   - 检查是否授予了bookmarks权限
   - 查看浏览器控制台错误信息

3. **通知不显示**
   - 确认浏览器允许通知权限
   - 检查notifications权限是否正确配置

### 调试方法

1. 打开Chrome扩展管理页面
2. 找到BookmarkManager扩展
3. 点击"详细信息"
4. 点击"检查视图"中的相关链接进行调试
